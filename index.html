ตั้งค่าแหล่งข้อมูล

วางลิงก์ CSV ที่ได้จากการ "เผยแพร่สู่เว็บ" ของชีต Transaction ใน Google Sheet

วิธีเอาลิงก์ CSV จาก Google Sheet
  1. เปิด Google Sheet → ไปที่แท็บ Transaction
  2. เมนู File → Share → Publish to web
  3. เลือกแท็บ "Transaction" และรูปแบบ "Comma-separated values (.csv)"
  4. กด Publish แล้วคัดลอกลิงก์ที่ได้มาวางในช่องด้านบน
  5. ทุกครั้งที่แก้ไขข้อมูลในชีต กลับมาหน้านี้แล้วกด "รีเฟรชข้อมูล"
